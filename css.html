CSS Anatomy

The anatomy of CSS applies to how CSS is incorporated within a site. Both these methods are the 'ruleset' and 'inline style'.

Let's look at the anatomy for both these applications:

Rulesets

Rulesets are styles which are defined either within the HTML file itself, or in a seperate style file with a .css extension. 

            
                /* Ruleset anatomy */
                selector {
                    property: value;
                }

                /* Example of a CSS ruleset */
                body {
                    background-color: white;
                    color: black;
                }
